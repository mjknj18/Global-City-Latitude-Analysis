Project Summary

The purpose of this assignment was to analyze current weather data from over 500 randomly selected global cities. The Numpy, Citipy, and Requests modules in Python were used to generate random latitude and longitude combinations, find the name of the nearest city for each combination, and extract weather data from the Open Weather Map API for each city. Python's Pandas and Matplotlib modules were used to process this data to find certain metrics and generate plots to display meaningful results. This facilitated several observations regarding how maximum temperature, humidity, wind speed, and cloudiness vary based on a city's latitude.
